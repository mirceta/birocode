3 Why “same box” is what unlocks the pointer

The only reason the paste was a copy was the fear the other agent couldn’t reach the convention. On-box, that fear is gone — for a simpler reason than HTTP:

  1. It reads the doc off disk. No server call at all — just open the file at its absolute path. Always works on-box.
  2. No WebFetch empty-shell problem. The homepage is JS-rendered, so fetching it over HTTP returns a blank page. A plain markdown file sidesteps that entirely.
  3. One source of truth. The contract lives in one file; the paste and the homepage tutorial both just point at it.
  4. No drift. Change the convention once and every agent — and the homepage — reflects it. No stale pasted copies to chase.
