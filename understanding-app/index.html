One new field on an idea

An idea is a Note. We add an optional Active boolean — the same no-migration pattern as project and priority (old entries default to false).

＋

No notes.json migration: System.Text.Json fills the missing field with its default, so existing ideas just read as not active.

      Scope & open questions

      In scope (v1)

      • Binary Active flag per idea
      • Active group pinned at the top + per-card toggle
      • Shows in the Ideas tab and dashboard panel
      • Filter narrows both groups

      Out of scope

      • Full todo lifecycle (doing / blocked / done)
      • Manual drag-ordering within Active
      • Any coupling to autopilot / agent docks

      Open questions (defaults chosen)

      • Empty Active header → hidden
      • Active × priority → independent
      • Collapsible sections → no
