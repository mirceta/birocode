Queued prompts

An agent can only run one prompt at a time. The queue (the former prompt-stash, now merged) lets you line up the next ones while it's busy. Nothing runs on its own — when the agent is free you approve each one (tap Send) to send it, or tap × to delete it. It lives on both the main chat and every agent tab, and survives refresh & redeploy.
