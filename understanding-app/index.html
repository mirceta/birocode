Hide inactive agents — “Show only important”

The dashboard shows every agent dock at once. This adds one switch: flip it on and every dock that isn’t starred ★ important disappears, leaving just the ones you care about. It’s a pure view filter on the star you already have. Drive it below.

How it’s built

✓ Reuses the star

The signal already exists: the backend-synced important flag on each dock (plans/important-agents.md). The switch just filters the dock list to those — no new dock data, no backend change.

✓ Device-local toggle

One boolean in localStorage, default off — same pattern as the dashboard’s size / zoom / layout prefs in Dashboard.jsx. A per-viewer view preference, not shared.

✓ Both surfaces

Phone docks and summary cards render the same ordered list, so the filter applies to both at once — nothing special-cased.

✓ Never looks broken

If the filter is on and nothing is starred, an explicit empty-state hint shows instead of a blank dashboard.
